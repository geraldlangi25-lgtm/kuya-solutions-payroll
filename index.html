Update the name and pay period dates for the currently selected schedule. This also fixes which billing month it appears under.
💡 The billing month is detected from the Pay Period Dates — make sure the dates contain the correct month name (e.g. "March 16–31, 2026").
☁ Cloud Sync Setup
Connect a free JSONBin.io storage bin so every teammate sees live changes from any device — no server needed.
👉 Step 1: Go to jsonbin.io → Sign up free → Dashboard → API Keys → copy your Master Key.
Step 2: Paste the key below, then click Create New Bin (first time) or enter an existing Bin ID.
Step 3: Click Save & Connect. Share the same Bin ID + Key with your team.
🔒 Privacy: Your bin is private by default — only people with your Master Key can read it. Keep the key secure and only share it with trusted team members. JSONBin free tier allows unlimited reads/writes for small payloads.
